# 3.1.2 Screen Descriptions

| # | Feature | Screen | Description |
|---|---|---|---|
| 1 | Authentication | Web Login Screen | Allows Admin and Librarian to sign in to the web system using SLIB credentials. |
| 2 | Authentication | Forgot Password Screen | Allows staff and mobile users to request password reset when they cannot access the system. |
| 3 | Authentication | Mobile OnBoarding Screen | Introduces Student and Teacher users to the main SLIB mobile features before login. |
| 4 | Authentication | Mobile Login Screen | Allows Student and Teacher users to log in to the mobile app using Google account or SLIB account. |
| 5 | Authentication | First-Time Password Change Screen | Requires users to update their password on first login when applicable. |
| 6 | Dashboard | Admin Dashboard | Provides Admin with an overview of the system and shortcuts to major administration modules. |
| 7 | Dashboard | Librarian Dashboard | Provides Librarian with monitoring widgets, alerts, operational summaries, and quick navigation to service modules. |
| 8 | Monitoring | Check-In / Check-Out Screen | Allows Librarian to monitor access logs, filter events, and review check-in or check-out activity. |
| 9 | Monitoring | Seat and Area Management Screen | Allows Librarian to observe area layout, seat states, and zone details for real-time supervision. |
| 10 | Booking Management | Booking Management Screen | Allows Librarian to review, filter, inspect, and process user booking records. |
| 11 | User Monitoring | Student Management Screen | Allows Librarian to browse student records and open detailed student information. |
| 12 | Violation Handling | Violation Management Screen | Allows Librarian to review violation cases, inspect details, and process follow-up actions. |
| 13 | Support Handling | Support Request Management Screen | Allows Librarian to review support requests and respond to users. |
| 14 | Complaint Handling | Complaint Management Screen | Allows Librarian to inspect complaints and accept or deny pending cases. |
| 15 | Feedback Handling | Seat Status Report Management Screen | Allows Librarian to inspect seat status reports and verify report results. |
| 16 | Feedback Handling | Feedback Management Screen | Allows Librarian to view user feedback details and mark feedback as reviewed. |
| 17 | Chat & Support | Chat Management Screen | Allows Librarian to manage active conversations with students and review chat details. |
| 18 | Statistics | Statistics Screen | Allows Librarian to view analytical charts, operational metrics, and AI-supported statistics. |
| 19 | Statistics | AI Analytics Panel | Displays AI-driven analytics such as prioritized students, behavioral signals, and warning actions. |
| 20 | News & Announcement | News and Announcement List Screen | Allows Librarian to manage published content, categories, and navigation to create, edit, and view flows. |
| 21 | News & Announcement | Create News Screen | Allows Librarian to compose a new news or announcement post. |
| 22 | News & Announcement | Edit News Screen | Allows Librarian to edit an existing news or announcement post. |
| 23 | News & Announcement | News Detail View Screen | Allows Librarian to view a full news or announcement article in detail mode. |
| 24 | News & Announcement | New Book List Screen | Allows Librarian to manage the list of newly introduced books. |
| 25 | News & Announcement | Create New Book Screen | Allows Librarian to create a new book introduction entry. |
| 26 | News & Announcement | Edit New Book Screen | Allows Librarian to update an existing new book entry. |
| 27 | Kiosk Content | Kiosk Slideshow Management Screen | Allows Librarian to manage kiosk slideshow images, status, order, preview, and upload actions. |
| 28 | Kiosk Content | Kiosk Slideshow Preview Screen | Allows Librarian to preview the slideshow as it appears on kiosk displays. |
| 29 | Attendance | Attendance Waiting Screen | Displays the attendance waiting interface used in the librarian web flow. |
| 30 | Account Management | Account Settings Screen | Allows Admin and Librarian to manage their own account settings and sign-out flow. |
| 31 | Layout Administration | Library Map Management Screen | Allows Admin to design and manage areas, zones, seats, amenities, and layout elements. |
| 32 | User Management | User Management Screen | Allows Admin to search, import, inspect, update, and remove system user accounts. |
| 33 | HCE Scan Station Management | HCE Scan Station Management Screen | Allows Admin to manage HCE station records, registration, status, and monitoring details. |
| 34 | NFC Tag Management | NFC Tag Management Screen | Allows Admin to map NFC tags to seats, verify scans, and manage bridge-related NFC operations. |
| 35 | Kiosk Device Management | Kiosk Device Management Screen | Allows Admin to manage kiosk device records, activation, tokens, and operational status. |
| 36 | System Configuration | System Configuration Screen | Allows Admin to configure library rules, reputation rules, notification settings, and backup scheduling. |
| 37 | System Monitoring | System Health Screen | Allows Admin to inspect service health, system logs, and backup execution history. |
| 38 | AI Configuration | AI Configuration Screen | Allows Admin to manage AI materials, knowledge stores, and AI testing tools. |
| 39 | Home | Home Tab | Displays the main mobile dashboard with booking summary, quick actions, AI suggestions, news, and new books. |
| 40 | Notifications | Notification Screen | Allows Student and Teacher users to view, filter, and inspect notification content. |
| 41 | Booking | Floor Plan Tab | Allows Student and Teacher users to browse areas, choose time slots, inspect seats, and start booking flows. |
| 42 | Booking | Booking Confirmation Screen | Allows Student and Teacher users to confirm seat booking details before completing a reservation. |
| 43 | Booking | QR Scan Screen | Allows Student and Teacher users to scan QR codes for check-in or check-out actions. |
| 44 | Library Card | Student / Teacher Card Tab | Displays the user’s digital library card and barcode information. |
| 45 | News & Announcement | News List Screen | Allows Student and Teacher users to browse all news and announcements. |
| 46 | News & Announcement | News Detail Screen | Displays the full content of a selected news or announcement item. |
| 47 | New Books | New Books List Screen | Allows Student and Teacher users to browse all newly added books. |
| 48 | New Books | New Book Detail Screen | Displays detailed information about a selected new book. |
| 49 | Chat & Support | Chat Tab | Provides Student and Teacher users with AI chat, escalation to librarian, and support-related conversation flows. |
| 50 | Support | Support Request Screen | Allows Student and Teacher users to submit support requests with optional image attachments. |
| 51 | Support | Support Request History Screen | Allows Student and Teacher users to review submitted support requests and their statuses. |
| 52 | Account Management | Account and Settings Tab | Allows Student and Teacher users to access personal settings, histories, support, and logout actions. |
| 53 | Profile | Profile Information Screen | Allows Student and Teacher users to view and edit their personal profile information. |
| 54 | Booking History | Booking History Screen | Allows Student and Teacher users to view booking history and open booking action details. |
| 55 | Activity History | Activity History Screen | Allows Student and Teacher users to review historical activity records. |
| 56 | Violation History | Violation History Screen | Allows Student and Teacher users to review violations, penalties, and complaint-related detail flows. |
| 57 | Complaint History | Complaint History Screen | Allows Student and Teacher users to review submitted complaints and complaint status details. |
| 58 | Report History | Report History Screen | Allows Student and Teacher users to review submitted seat status and violation reports. |
| 59 | Reporting | Seat Status Report Screen | Allows Student and Teacher users to submit a seat status issue report. |
| 60 | Reporting | Seat Violation Report Screen | Allows Student and Teacher users to submit a seat violation report. |

## Notes

- The descriptions are derived from the latest Admin website, Librarian website, and Student/Teacher mobile screen flow diagrams.
- Both route-based screens and major standalone mobile screens are included here.
- Popups, bottom sheets, and confirm dialogs are represented in the flow diagrams, while this document focuses on primary named screens.
