# User Management Module

This folder contains the Report 4 diagrams for Module 3 - User Management.

## Included Artifacts

- Sequence diagrams for:
  - `FE-15` View list of users in the system
  - `FE-16` Import Student and Teacher via file
  - `FE-17` Download template of the file upload
  - `FE-18` Add Librarian to the system
  - `FE-19` View user details
  - `FE-20` Change user status
  - `FE-21` Delete user account
- Class diagram for the User Management Module

## Actor Scope

- `FE-15` to `FE-21`: `Admin`

## Sequence Diagram Convention

- Step numbering is written explicitly in each message.
- `alt` branches are used for different outcomes or current implementation variants.
- The `Users` column follows the Module 1 and Module 2 convention for `mermaid.ai` rendering.
- Numbering continues correctly after each `alt` block.
