# Authentication Module

This folder contains the Report 4 diagrams for Module 1 - Authentication.

## Included Artifacts

- Sequence diagrams for:
  - `FE-01` Log in via Google Account
  - `FE-02` Log in via SLIB Account
  - `FE-03` Forgot password
  - `FE-04` Log out
- Class diagram for the Authentication Module

## Sequence Diagram Convention

- Step numbering is written explicitly in each message.
- `alt` branches are used for different outcomes or platform-specific flows.
- Numbering continues correctly after each `alt` block.
