# Module 12: Statistics & Report

## 12.1 Statistics Management

### FE-115: View general analytics dashboard (AI-enhanced)

- **Actors:** Librarian, Admin
- **Navigation path:** Sidebar -> "Tổng quan" / Dashboard
- **Purpose:** View overview statistics dashboard with AI insights
- **Interface:**
    1. **Key metrics cards:** Visitors, bookings, occupancy (shows "Chỗ ngồi đã có người")
    2. **Charts:** Trends over time
    3. **AI Insights:** AI analysis and predictions
    4. **"Khu vực được sử dụng nhiều nhất":** Most popular zone card
    5. **Comparisons:** Week-over-week, month-over-month
- **Data:** Aggregated metrics, AI-generated insights

---

### FE-116: View violation statistics

- **Actors:** Librarian, Admin
- **Purpose:** View violation statistics by time, type
- **Interface:** Charts with filters
- **Data:** Violation counts by type, time period, "Khu vực"
- **Charts:** Pie chart (by type), Line chart (trend)

---

### FE-117: View statistics of density forecast by using AI

- **Actors:** Librarian, Admin
- **Purpose:** View AI density predictions
- **Interface:** Prediction heatmap by hour/day
- **Data:** Predicted occupancy by hour, by day
- **AI Features:**
    - Peak hour prediction
    - Crowding alerts
    - Optimal booking time suggestions

---

### FE-118: View check-in/check-out statistics (Daily/Weekly/Monthly)

- **Data:** Check-in/out counts by period
