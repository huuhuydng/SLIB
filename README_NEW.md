# SLIB - README Draft

## Overview
SLIB is a multi-service system with:
- Backend (Spring Boot)
- Frontend (Vite + React)
- Mobile (Flutter)
- AI Service (FastAPI)

## Project Structure
- `backend/`: Java Spring Boot API
- `frontend/`: React web app
- `mobile/`: Flutter app
- `ai-service/`: Python FastAPI service

## Quick Start
### 1) Backend
```bash
cd backend
./mvnw spring-boot:run
```

### 2) Frontend
```bash
cd frontend
npm install
npm run dev
```

### 3) AI Service
```bash
cd ai-service
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### 4) Mobile
```bash
cd mobile
flutter pub get
flutter run
```

## Notes
- This file is a draft README created for preview.
