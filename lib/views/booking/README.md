# Booking Feature

Tính năng đặt chỗ ngồi trong thư viện.

## Files
- `booking_screen.dart` - Màn hình đặt chỗ chính

## Widgets (Sẽ phát triển)
- `room_selector.dart` - Widget chọn phòng
- `floor_selector.dart` - Widget chọn tầng
- `seat_grid.dart` - Lưới ghế ngồi
- `time_picker.dart` - Chọn thời gian đặt
- `booking_summary.dart` - Tóm tắt đặt chỗ

## Features
- Chọn phòng/tầng
- Xem sơ đồ ghế real-time
- Chọn thời gian đặt
- Xác nhận đặt chỗ
