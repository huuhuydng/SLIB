# Check-in Feature

Tính năng check-in vào thư viện bằng QR code.

## Files
- `checkin_screen.dart` - Màn hình check-in chính

## Widgets (Sẽ phát triển)
- `qr_scanner.dart` - Widget quét QR code
- `checkin_success.dart` - Dialog thành công
- `checkin_history.dart` - Lịch sử check-in

## Features
- Quét QR code để check-in
- Xác nhận vị trí
- Ghi nhận thời gian
- Hiển thị trạng thái check-in
