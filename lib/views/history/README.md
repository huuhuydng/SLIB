# History Feature

Tính năng xem lịch sử hoạt động.

## Files
- `history_screen.dart` - Màn hình lịch sử

## Widgets (Sẽ phát triển)
- `history_filter.dart` - Bộ lọc lịch sử
- `history_item.dart` - Item lịch sử
- `history_timeline.dart` - Timeline lịch sử
- `stats_card.dart` - Thống kê

## Features
- Xem lịch sử đặt chỗ
- Xem lịch sử check-in
- Lọc theo ngày/loại
- Thống kê hoạt động
