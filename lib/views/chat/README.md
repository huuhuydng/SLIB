# Chat Feature

Tính năng chat/hỗ trợ.

## Files
- `chat_screen.dart` - Màn hình chat

## Widgets (Sẽ phát triển)
- `chat_message.dart` - Bubble tin nhắn
- `chat_input.dart` - Input gửi tin nhắn
- `chat_header.dart` - Header chat

## Features
- Chat với quản trị viên
- Hỗ trợ khách hàng
- Gửi file/hình ảnh
- Thông báo real-time
