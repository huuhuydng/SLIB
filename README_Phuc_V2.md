# SLIB Project Documentation

## 1. Introduction
This project is a library management system designed to support students, librarians, and administrators.

## 2. Features
- User authentication (Google, SLIB account)
- Book management
- Borrow and return system
- User management
- Reporting and statistics

## 3. Technologies Used
- Backend: Spring Boot
- Frontend: ReactJS
- Database: MySQL
- Authentication: OAuth2

## 4. Project Structure
- backend/
- frontend/
- database/
- docs/

## 5. Installation
1. Clone the repository
2. Install dependencies
3. Configure database
4. Run backend server
5. Run frontend app

## 6. Usage
Users can log in and perform actions based on their roles.

## 7. API Overview
- GET /api/books
- POST /api/books
- PUT /api/books/{id}
- DELETE /api/books/{id}

## 8. Database Design
Includes tables for users, books, transactions, and roles.

## 9. Authentication
Supports Google login and internal SLIB accounts.

## 10. Contribution
Pull requests are welcome. Please follow coding standards.

## 11. License
This project is licensed under MIT License.

## 12. Contact
For any issues, please contact the development team.# SLIB Project Documentation
