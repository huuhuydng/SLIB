# Map Feature

Tính năng xem sơ đồ thư viện.

## Files
- `map_screen.dart` - Màn hình sơ đồ thư viện

## Widgets (Sẽ phát triển)
- `floor_map.dart` - Bản đồ từng tầng
- `room_info.dart` - Thông tin phòng
- `seat_status.dart` - Trạng thái ghế
- `legend.dart` - Chú thích bản đồ

## Features
- Xem sơ đồ các tầng
- Xem trạng thái ghế real-time
- Tìm đường đến vị trí
- Xem thông tin phòng
